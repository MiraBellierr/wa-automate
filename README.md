> A Whatsapp bot written in NodeJS by me to automate tasks.

## Install

```sh
npm install
```

## Usage

Make sure to rename the `.env.example` to `.env` and fill the information

```
PREFIX=
OPEN_AI_KEY=
```

To start this project, run the following command:

```sh
node .
```
